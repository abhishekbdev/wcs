The Library of the college was established in1972 along with the opening of the college. The separate spacious building for library was built in 1985. The Central Library holds rich collection of books, journals, periodicals and online database. The Library is having spacious building well equipped with furniture, book racks, computers, printers, Internet connection and UPS power back-up. Specious well-ventilated illuminated reading hall with 60 seating capacity. Ample space for stack room and administrative work. The library is equipped with various efficient and well organised sections; viz. Acquisition Section, Processing Section (Book prossessing, data entry, barcoding, shelving, etc.), Periodical Section, Circulation Section, Reading Section, Internet Section, Reference Section, Stack Room. The library is automated. The library has Web-OPAC to search the bibliographic database. Internet facility is made available with 5 nodes in the digital library. Students and faculty can access N-LIST database having e-books, e-materials in the library. Access to the database is also provided to the departments having Internet facility as per the authentication policy of the Database.
